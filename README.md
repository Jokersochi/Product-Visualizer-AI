# 🖼️ Product Visualizer AI

Это приложение для визуализации продуктов с использованием **React**, **TypeScript** и **Google Gemini API** для генерации изображений.

## 🚀 Улучшения в этом Pull Request

*   **Улучшена структура проекта:** Добавлены файлы `.gitignore` и `.nvmrc` для лучшей совместимости и управления зависимостями.
*   **Актуализация документации:** Обновлены инструкции в `README.md`.

## 🛠 Запуск проекта локально

**Предварительные требования:**
*   [Node.js](https://nodejs.org/en/) (рекомендуется LTS)
*   [npm](https://www.npmjs.com/) или [pnpm](https://pnpm.io/)

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Jokersochi/Product-Visualizer-AI.git
    cd Product-Visualizer-AI
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    # или pnpm install
    ```

3.  **Настройте ключ API:**
    *   Создайте файл `.env.local` в корне проекта.
    *   Установите ваш ключ Gemini API:
        ```
        VITE_GEMINI_API_KEY="ВАШ_КЛЮЧ_API"
        ```

4.  **Запустите приложение:**
    ```bash
    npm run dev
    # или pnpm run dev
    ```

Приложение будет доступно по адресу `http://localhost:5173` (или другому порту, указанному в консоли).
